flatMap

abstract fun <K> flatMap(mappingFunction: suspend (T) -> SuspendStream<K>): SuspendStream<K>

Returns a stream consisting of the results of replacing each element of this stream with the contents of a mapped stream produced by applying the provided mapping function to each element. Each mapped stream is closed after its contents have been placed into this stream. This is an intermediate operation.
