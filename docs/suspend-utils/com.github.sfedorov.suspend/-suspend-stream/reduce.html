abstract suspend fun <U> reduce(unit: U, mergeFunction: suspend (U, T) -> U): U

Performs a reduction on the elements of this stream, using the provided identity value and an accumulation function, and returns the reduced value. Elements are ordered. This is a terminal operation.

abstract suspend fun reduce(mergeFunction: suspend (T, T) -> T): T?

Performs a reduction on the elements of this stream, using an associative accumulation function, and returns the reduced value, if any, or null otherwise. Elements are ordered. This is a terminal operation. Null can be returned also in case when stream type allows null elements and null is the result of reduction.
