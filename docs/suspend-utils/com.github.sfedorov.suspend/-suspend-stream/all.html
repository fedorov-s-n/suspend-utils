all

abstract suspend fun all(predicate: suspend (T) -> Boolean): Boolean

Returns whether all elements of this stream match the provided predicate. Should not evaluate the predicate on all elements if not necessary for determining the result. If the stream is empty then true is returned and the predicate is not evaluated. This is a short-circuiting terminal operation.
